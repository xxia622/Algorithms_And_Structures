## Technical Focus
I adhere to these core principles in every implementation:
1. **Performance First**: Every line of code is scrutinized for **Time/Space Complexity**, striving for $O(n \log n)$ or better efficiency.
2. **C++ Best Practices**: Strictly following memory management standards, utilizing pointers and modern C++ features for low-level optimization.
3. **Readability**: Maintaining professional naming conventions and clear logic documentation, even for competitive programming tasks.
